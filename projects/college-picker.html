College Picker

Full-stack web app that matches users to colleges based on weighted preferences. Evolved through three complete iterations — from a first internship prototype on Azure, to a team class project on Google Cloud, to a cleaner solo production version on Supabase.

Background

Built during a first internship at Aneyeon, Inc., a local custom software development company. The project was personally motivated — the college application process is overwhelming, and a weighted-preference recommender seemed like a genuinely useful tool. Two years later, it was adapted and expanded for CS 411 (Database Systems) at UIUC as a team project, with heavier SQL requirements. It was later rewritten solo as a cleaner personal project.

Objective

Ask users a series of questions about college preferences (state, size, religious affiliation, test scores, etc.) and how important each criterion is to them. Generate a ranked top-5 college list based on weighted preferences, with full detail pages for each match.

Outcome

A live, deployed web application demonstrating the full product lifecycle — from intern prototype to a production-ready personal project. Currently hosted on Google Cloud Run with a PostgreSQL database on Supabase.
