# Artem Khaiet — Portfolio

Personal portfolio site built with vanilla HTML/CSS/JS. Hosted on GitHub Pages.

## Structure

```
portfolio/
├── index.html          # Homepage with project grid + filters
├── css/
│   └── style.css       # All styles
├── js/
│   └── data.js         # Project data + shared components (nav, footer)
└── projects/

    ├── college-picker.html
    ├── sigaida-dashboard.html
    ├── calorie-tracker.html
    ├── county-affordability.html
    ├── sports-spending.html
```
